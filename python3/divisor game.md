# Divisor Game

#### Problem Link: https://leetcode.com/problems/divisor-game/

## Approach 1

### Solution:

```py
class Solution:
    def divisorGame(self, n: int) -> bool:
        if n%2 == 0:
            return True
        else:
            return False

```
